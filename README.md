# Xjapan-Project
<코로나 감염자 지도 API 서비스>

1.주요 기능
내 현재 위치를 검색할 수 있다.
현재 날짜를 지도위에 표시할 수 있다.
감염자가 발생하는 지역을 지도위에 표시할 수 있다.
같은 지역에 n명 이상의 사람들이 존재할 경우 Cluster화 하여 표시한다.
2. 활용 Tool
Node. js / npm
Mongo DB
JavaScript
